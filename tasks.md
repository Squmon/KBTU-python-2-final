# Формат Векторов
Т.к использовать numpy нельзя, вектора будут списоком вида:

Ключи - название оси
Значение - значение проекции вектора на соответствующую ось

При сложении/скалярном произведении/поэлентных операциях соответствующие ключи будут складыватся, а при несовпадении ключей будет предполагатся что отсутствующие ключи имеют значение ноль

Пример:
```
A = vector(
    x = 3,
    y = 2
) 

B = vector(
    x = 1,
    y = 10,
    z = -1
)

A + B -> vector(
    x = 4
    y = 12
    z = -1
)

# @ - это скалярное произведение
A @ B -> 3*1 + 2 * 10 + -1 * 0 = 23
A * B -> vector(x = 3*1, y = 2*10, z = -1*0)
A / B -> vector(x = 3/1, y = 0.2, z = 0)
B / A -> error, т.к предполагается что в A['z'] = 0
```
